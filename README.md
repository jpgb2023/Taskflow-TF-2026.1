# 📌 TaskFlow

Sistema de gerenciamento de tarefas desenvolvido como projeto acadêmico na disciplina de Desenvolvimento Ágil do curso de Engenharia da Computação.

---

## 🚀 Sobre o Projeto

O **TaskFlow** é uma aplicação simples de lista de tarefas (To-do List) que permite aos usuários organizar suas atividades diárias de forma prática e eficiente.

O projeto foi desenvolvido com foco na aplicação de conceitos de metodologias ágeis, como Scrum e XP, além de utilizar conhecimentos adquiridos nas disciplinas de Programação Orientada a Objetos, Banco de Dados e Algoritmos.

---

## 🎯 Objetivo

Facilitar a organização de tarefas do usuário, promovendo maior produtividade e controle das atividades do dia a dia.

---

## ⚙️ Funcionalidades

- ✅ Criar tarefas  
- 📋 Listar tarefas  
- ✔️ Marcar tarefa como concluída  
- ❌ Excluir tarefa  
- 🔍 Filtrar tarefas (pendentes/concluídas) *(opcional)*  
- ⭐ Definir prioridade *(opcional)*  

---

## 🛠️ Tecnologias Utilizadas

- Linguagem: (HTML + CSS + JavaScript)
- Paradigma: Programação Orientada a Objetos (POO)
- Banco de Dados: (MySQL)
- Metodologia: Scrum / XP
