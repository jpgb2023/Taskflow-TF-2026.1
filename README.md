## 🚀 Sobre o Projeto

O **TaskFlow** é um sistema de gerenciamento de tarefas (To-do List) desenvolvido com o objetivo de auxiliar usuários na organização de suas atividades diárias de forma simples, prática e eficiente.

O sistema permitirá o controle de tarefas por meio de uma interface intuitiva, facilitando o acompanhamento das atividades e aumentando a produtividade do usuário.

### 🎯 Objetivo
Proporcionar uma ferramenta simples e eficiente para organização de tarefas, aplicando conceitos de desenvolvimento ágil e boas práticas de programação.

### ⚙️ Principais Funcionalidades
- Criar tarefas  
- Listar tarefas  
- Marcar tarefas como concluídas  
- Excluir tarefas  
- Filtrar tarefas por status *(opcional)*  
- Definir prioridade *(opcional)*  

### 👥 Público-Alvo
Usuários que desejam organizar suas tarefas diárias de forma simples, como estudantes, profissionais e pessoas que buscam melhorar sua produtividade.
