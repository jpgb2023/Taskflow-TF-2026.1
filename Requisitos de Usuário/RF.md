# 1. Requisitos Funcionais

| RF01 |              O usuário deve poder criar novas tarefas.                    |    Alta    |           -             |

| RF03 |        O usuário deve poder marcar tarefas como concluídas.               |    Alta    |         RF02            |
| RF04 |              O usuário deve poder excluir tarefas.                        |    Alta    |         RF02            |
